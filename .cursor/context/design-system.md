# Design System Snapshot

Tokens (use Tailwind classes):
- Brand surfaces: bg-background, text-foreground
- Primary: bg-primary, text-primary-foreground
- Secondary: bg-secondary, text-secondary-foreground
- Muted: text-muted-foreground
- Card: bg-card, text-card-foreground, border-border

Typography:
- Headings bold, text-foreground
- Body text-muted-foreground
- Max line length ~65–75ch

Layout:
- Section spacing py-16–py-24
- Container widths via Tailwind container utilities
- Rounded corners: rounded-lg / rounded-xl
- Subtle shadows on interactive surfaces

Component guidance:
- Buttons: shadcn Button (variants: default, secondary, outline, ghost)
- Cards: shadcn Card for surfaces
- Forms: Input, Label, Select, Textarea
- Disclosure: Accordion/Collapsible for services/FAQ
- Badges, Separator, Avatar as needed
